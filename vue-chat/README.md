# Vue 3 Chat Realtime

Implementación en Vue 3 para la prueba técnica de Chat Realtime.

## 🚀 Inicio rápido

1. **Configuración**: Asegúrate de tener un archivo `.env` basado en `.env.example`.
2. **Instalación**: `npm install`
3. **Ejecución**: `npm run dev`
